# Cathy DOLLE - Bubble Tea Maker

## Features
You can do your own bubble tea, choose the :
- size
- label
- flavor
- topping
- straw

You can click on "post-it"
- NJS (link -> game)
- 3D shop
- My logo + grid
- Recipe
- custom note

Music player: switch, pause, play

- Night / Day mode
